# Ethstat-parity

To use:

install packages:
`npm install`

update dist folder:
`webpack`


link the dist with parity

For Mac
`ln -s $PWD/dist $HOME/Library/Application\ Support/io.parity.ethereum/dapps/mydapp`

For Linux
`ln -s $PWD/dist $HOME/.local/share/io.parity.ethereum/dapps/mydapp`
